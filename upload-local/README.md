# Upload Single

## Descrição
API para fazer upload de arquivos e armazenar localmente na pasta [uploads.]('tmp/uploads')
- Dependências:
    - Express.
    - Morgan.
    - Mongoose.
    - Multer.

## Upload
- Rota que aceita multipart/form

> POST /

    Só são aceitos arquivos do tipo (PNG, JPEG E JPG).
