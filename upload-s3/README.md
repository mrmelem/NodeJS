# Upload para s3

## Descrição
API para fazer upload de arquivos e armazenar na AWS s3.
- Dependências:
    - Aws Sdk
    - Multer s3
    - Express.
    - Morgan.
    - Mongoose.
    - Multer.

## Upload
- Rota que aceita multipart/form

> POST /

    Só são aceitos arquivos do tipo (PNG, JPEG E JPG).
